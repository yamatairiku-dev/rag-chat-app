## 🚀 最速セットアップ（5分）

今すぐ始めたい方は、以下をコピペ実行：

```bash
# 1. プロジェクト作成
npx create-react-router@latest rag-chat-app
cd rag-chat-app

# 2. setup.sh実行
cp ~/Downloads/setup.sh .
chmod +x setup.sh
./setup.sh

# 3. ファイルコピー
cp ~/Downloads/*.md docs/
cp ~/Downloads/.env.* .

# 4. .env作成（最小限の設定）
cp .env.development.example .env

# エディタで.envを開いて以下を設定:
# - ENTRA_CLIENT_ID
# - ENTRA_CLIENT_SECRET
# - ENTRA_TENANT_ID
# - DIFY_API_URL
# - DIFY_API_KEY
# - SESSION_SECRET (node -e "console.log(require('crypto').randomBytes(32).toString('hex'))" で生成)

# 5. 依存関係インストール
npm install
npx shadcn@latest init
npx shadcn@latest add button input card avatar scroll-area separator

# 6. 起動！
npm run dev
```

詳細は **クイックスタートガイド.md** を参照。
